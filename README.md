EcoRide – Smart Carpooling Platform
Description

EcoRide is a smart carpooling web platform designed to simplify transportation between drivers and passengers through an intuitive and modern interface.

The platform allows users to search for trips, reserve seats, manage vehicles, report issues, and interact with different services related to transportation and events.

The project was developed using an MVC architecture with advanced functionalities such as AI-powered search tools, trip visualization on maps, statistics management, and event administration.

Architecture

This project follows the MVC (Model - View - Controller) architecture:

Model → Handles database operations and business logic
View → Manages the user interface and pages
Controller → Connects models and views while handling application logic

Installation
1. Clone the repository
git clone https://github.com/your-username/ecoride.git
2. Move the project into your local server directory

Example for XAMPP:

htdocs/ecoride
3. Import the database

Import the SQL file into phpMyAdmin.

4. Configure database connection

Edit:

config/database.php
5. Start Apache & MySQL

Launch XAMPP or your local server.

6. Run the project
http://localhost/ecoride
