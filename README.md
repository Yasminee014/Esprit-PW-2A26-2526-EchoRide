# EcoRide – Système de Gestion de Transport

## Aperçu

Ce projet a été développé dans le cadre du module PIDEV – Cycle Ingénieur 3ème année à **Esprit School of Engineering** (Année universitaire 2025–2026).
Il s’agit d’une application web permettant de gérer un système de transport incluant les utilisateurs, les trajets, les véhicules, les colis, les notifications ainsi que les objets perdus.

La plateforme permet une gestion efficace des opérations de transport et des interactions entre les utilisateurs à travers plusieurs modules fonctionnels.

---

## Technologies utilisées

* HTML / CSS
* JavaScript
* PHP
* MySQL

---

## Modules

* **Gestion des utilisateurs** (users, rôles)
* **Gestion des trajets** (trajets, destinations)
* **Gestion des colis** (colis, livraison)
* **Système de notification** (notifications, messages)
* **Objets perdus (Lost & Found)** (objets perdus, signalements conducteur)
* **Gestion des véhicules** (véhicules, réservations)

## Installation

### 1. Cloner le repository

```bash id="p8z7cm"
git clone https://github.com/username/ecoride.git
cd ecoride
```

### 2. Configurer l’environnement

* Installer un serveur local (XAMPP ou WAMP)
* Placer le projet dans le dossier `htdocs`

### 3. Importer la base de données

* Ouvrir phpMyAdmin
* Créer une base de données
* Importer le fichier `.sql`

### 4. Lancer le projet

```bash id="b7kqsm"
php -S localhost:8000
```
